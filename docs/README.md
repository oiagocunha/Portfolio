# Portfólio — Iago Cunha

Portfólio orientado a **engenharia de backend e sistemas distribuídos**: orquestração, mensageria, automação, dados e operação em produção.

## Posicionamento

Este repositório não é uma vitrine genérica de projetos pessoais. Ele documenta como penso e opero sistemas reais:

- problemas técnicos e tradeoffs
- decisões arquiteturais
- resiliência, desacoplamento e observabilidade
- processamento assíncrono e integração entre serviços

## Destaques

- Case study de plataforma multi-serviço com Temporal, Kafka, RAG e LDAP
- Visualização interativa de pipeline assíncrono
- Cards de projeto com complexidade, impacto e contexto técnico
- Conteúdo multilíngue (`pt`, `en`, `es`)
